Paged Design

Customisable themes for book design with CSS

Are you creating books with HTML and CSS? Stylesheets for books can be long and complicated to write. Our themes give you a head start: pick a theme, add it to your project, and customise it.

Note that these demos only work in a Chrome browser, and that paged.js and this project are in development, so expect bugs, workarounds, and exciting possibilities, and contribute if you can.

How to use a theme

Setting up

  1. You'll need to have NodeJS installed on your computer. Our themes are written in Sass, and you'll use Node to build that into your finished CSS.
  2. Download a theme package and unzip it.
  3. Place the unzipped theme folder in your project. We're assuming that your project is contained in a single folder (like a Git repo), in which you can add your theme folder.
  4. In the head of your HTML (the content of your book), link to the compiled CSS file in your new theme folder. The path will depend on your folder structure, for example ../css/beatrix/themes/beatrix/main.css:
    <link rel="stylesheet" type="text/css" href="../css/beatrix/themes/beatrix/main.css">
  5. In the head of your HTML, link to the paged.js script, which will lay out your pages in Chrome. For instance, if you want to always link to the latest official version: 
    <script src="https://unpkg.com/pagedjs/dist/paged.polyfill.js"></script>
  6. Serve your HTML. For example, if you're using Node and have http-server installed, you can run npm run http-server -p 4000 in your project root folder, and visit your site at http://localhost:4000.
Now that you're up and running, you can customise your theme.

Customising a theme

Once you're set up:

  1. Install dependencies. To do this, from the command line, in the unzipped theme folder (alongside package.json), enter:

    npm install

    which will install a modules to node_modules that we need in that folder.

  2. Start our Node script to watch for changes to your styles. To do this, in the same folder, at the command line, enter:

    npm buildcss

    Node Sass will now be watching for changes to your styles. Every time you change a file in your theme's Sass files, your compiled CSS will automatically update and you only need to refresh your browser to see your new styles. You may need to hard refresh (Ctrl+F5 or Cmd+R).

  3. Set your selectors for book features in /themes/YOURTHEME/_selectors.scss (where YOURTHEME is the name of the theme youâ€™ve downloaded).
  4. Set any new values for design variables in /themes/YOURTHEME/_variables.scss. See the _defaults for the variables you can set.
  5. Add your own Sass/CSS styles to /themes/YOURTHEME/_styles.scss.

There are some technical docs on GitHub.
